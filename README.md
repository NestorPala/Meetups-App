# Meetups-App


__Este proyecto se ha desarrollado utilizando:__

* Node v16.17.0
* Express v4.18.2
* MongoDB (utilizando la librería Mongoose v6.6.5)
* React v18.2.0


__Para el Backend, además, he utilizado las siguientes librerías:__

* Nodemon v2.0.20
* Dotenv v16.0.3
* Cors v2.8.5
* Axios v1.1.2
